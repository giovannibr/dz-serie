# DZ Série

Um quiz que diz que série de TV você é baseado em como você age em algumas situações do dia-a- dia.

## Instalação
```
composer install
```
## Execução

### Método 1
```
composer start
```
### Método 2
```
php -S localhost:8080 -t public
```
### Método 3
Independende do servidor PHP, basta rodar apontando para a pasta "public"
